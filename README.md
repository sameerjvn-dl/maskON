# maskON

Mask Classifier that detects whether a face has a mask on or off. Fine tuned on a pretrained RESNET-18 model for 100% accuracy on validation set.

Deployed at - https://mask-on-clf.herokuapp.com/

Dataset available at - https://drive.google.com/drive/folders/1WPct1tTUFw5oJAmmu303A-kCZ_mTpm4B?usp=sharing
